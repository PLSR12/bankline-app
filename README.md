## :house: About the project
Aplicação completa de uma concessionária.

Na criação da API foi utilizado Express, JWT, yup, a biblioteca uuid, e o Sequelize para fazer a conexão com os bancos de dados que são o Postgres .

Na parte da tela do administrador , foi utilizado a biblioteca MUI para o uso dos ícones e criação das tabelas.
